Introduction

Sur cette page vous allez pouvoir retrouver les explications détaillées du jeu imagées. En effet le Skyjo est un jeu de société créée par Alexander Bernhardt en 2015 qui se joue de 2 à 8 joueurs. Il est aujourd'hui très répandu, certains le décrivent même comme le nouveau UNO. Le jeu connait aujourd'hui plusieurs variantes (Junior, action...) que nous pourrions imaginer dans un prolongement de notre version codée.

Etapes de programmation

La première étape a été de choisir notre langage pusique nous avions le choix entre le C++ et le Python. Si le C++ nous permettait d'entretenir nos connaissances développées cette année et de nous entrainer sur ce nouveau langage, notre choix s'est tout de même porté sur le Python car son aspect plus permissif nous semblait plus adapté pour faire un jeu.
De plus comme nous sommes en DLMI nous avons considéré le fait de devoir changer sans cesse de langage si le choix se portait sur le C++.
Pour programmer le jeu nous avons donc utilisé la bibliothèque Pygame pour tout ce qui concerne l'interface graphique et les intéractions utilisateurs-ordinateur, nos connaissances liées aux classes et aux différentes fonctionnalités de Python: voir l'arborscence des classes


Une fois le jeu terminé, il peut être intéressant de jouer contre un ordinateur.
Celui-ci utilisera différentes stratégies pour gagner la partie. Notamment, il peut choisir de prendre les cartes plus petites que celles de son jeu,
ou bien essayer de faire des colonnes, ou bien jouer complètement aléatoirement...
